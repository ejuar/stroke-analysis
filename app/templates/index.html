Introduction

A stroke is a medical emergency where blood flow to the brain is dangerously restricted resulting in brain cell death. According to a 2018 study in the New England Journal of Medicine, one in four people over the age of 25 will have a stroke in their lifetime. If not treated properly, aftereffects may include restricted motion, paralysis, and difficulty communicating. The purpose of this project is to raise awareness of strokes and the risk factors of strokes. This project uses a dataset from Kaggle which contains features that may be used to predict strokes. Some features analyzed in the dataset related to strokes are BMI, average glucose level, gender, being married, and age.

Data Cleaning

Why clean data?

The most important factor in data science is data. No matter what sophisticated machine learning models are used, the models are only as good as the data. Given a dataset, there are usually false data points and missing data that could compromise the accuracy of our modeling. Cleaning data helps identify and remove errors in the data.

Missing Value Handling

In our dataset, "BMI" was the only feature with missing data. We imputed this column by filling the missing values with the mean of "BMI".

Outlier Handling

All of the numerical data in our dataset had some outliers. However, we left them in because they were valid measurements, not impossible ones.

Imbalanced Data Handling

Imbalanced data occurs when one of the target classes make up a small proportion of the data. Out data is imbalanced, so we used SMOTE to balance our data. SMOTE balances data by synthesizing data with minority targets.

Machine Learning

Machine learning (ML) is a field of study that uses mathematical algorithms and data to make predictions and enhance the performance of tasks. Machine learning is considered to be a component of Artificial Intelligence (AI). Machine learning models "learn" by using data to tune hyperparameters in the machine learning model. Machine learning is utilized in a wide range of applications including speech recognition, email filtering, computer vision, and medicine. For our dataset, the target column "strokes" only has two class, having a stroke and not having a stroke. Therefore, we explored machine learning algorithms for binary classification.

Logistic Regression

Logistic regression was the first binary classifier we used in our project. Logistic regression produces a probability (whether something will or will not happen) which is compared to a threshold; if the probability is above the threshold, it is considered a positive case. For our dataset, logistic regression had a low accuracy of 54%. However, the recall rate was high (98%). Recall is the number of true positives divided by the sum of true positives and false negatives.

Decison Tree Classifier

The Decision Tree Classifier was created around the 1960s, however a similar concept has been used since the earliest written records. A decision tree classifier is a supervised machine learning algorithm that uses a tree like structure to categorize outcomes based on the answers to a previous set of questions. In terms of accuracy, the Decision Tree Classifier performed better than the logistic regression with an accuracy of 80%. However, the recall rate was low (49%).

Multi-Layer Perceptron

Multi-Layer Perceptrons, or neural networks, are another machine learning algorithm that use connections, activation functions, and layers to predict an output. Multi-layer perceptrons can function as a nonlinear function approximator for either classification or regression. The study of neural networks is a subset of machine learning called deep learning. Deep learning can make predictions on advanced tasks like self-driving cars and conversational virtual robots like Siri. Regarding predictions on our stroke dataset, the multi-layer perceptron model had the highest accuracy (88%), but the lowest recall (28%).

Machine Learning Results

The applicability of our models depend on the use case for stroke diagnoses. If one wants to use a model with very high accuracy, a null predictor (predicting everyone to not have a stroke) will be correct 96% of the time. However, if one wants to capture the population at higher risk for a stroke, the logistic regression model performed best with a 0.98 recall in the 'has had a stroke' category.

Conclusion

From exploratory data analysis, we observed that the most impactful stroke factors were age, average glucose level, and marriage status. The percentage of individuals that have a stroke increases after the age of 40. Additionally, individuals with higher average glucose levels and married individuals both have an increased risk of having a stroke.

Regarding model predictions, it was difficult to improve the accuracy past the null accuracy for this dataset. It was relatively small (5000 samples) with a highly imbalanced target distribution (about 5% minority class). Since models perform only as well as the quality of the data, the main improvement for future experiments in stroke prevention would be to increase quantity and diversity of data.
